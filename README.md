# DroneSwarm

App for OnTour drone swarm

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### for running in the demo (for tablet)

```sh
npm run dev -- --host
```

### For running the dummy backend:

(new cmd)

```sh
cd dummyEndSide
python server.py
```
